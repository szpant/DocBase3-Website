    Print Daily Worksheet
    Print Daily Worksheet
      • From SBS home page select Reports
      • From the drop down box select Print Worksheet/Checklist
      • Select Report Type: Daily Worksheet
      • Select Product Group: Driving Examination or Vehicle Inspection
      • Select Location from the drop down list
      • Enter Start & End date
      • Click on generate button
      Message Operation successful displays
      • Click on Click here to view report
      Adobe will open and report will default to landscape

      To print, use one of the following:
      • Click on the print icon
      • Select File, Print
      • Press Ctrl+P
    Print Vehicle Checklist
    Print Vehicle Checklist
      • From SBS home page select Reports
      • From the drop down box select Vehicle Checklist
      • Select Report Type: Vehicle Checklist
      • Select Location from the drop down list
      • Enter Start & End dates
      • Click on generate button
      Message Operation successful displays
      • Click on Click here to view report

      To print, use one of the following:
      • Click on the print icon
      • Select File, Print
      • Press Ctrl+P

      Helpful Hints
      A blank Vehicle Checklist can be print the same way by selecting "Blank Vehicle Checklist" in the Report Type

    Bulk Booking Reports
    Bulk Booking Reports
      • From the SBS home page select Reports
      • From the drop down box click on Bulk Booking
      • Select Product Group, Region and Location from the drop down lists
      • Select the Bulk Booking Name from the drop down list that you wish to display
      • Tick Status Type required : Bulk, Booked, Cancelled or all
      • Enter Start & End dates
      • Click on the generate button
      Message Operation successful displays
      • Click on Click here to view report

      To print, use one of the following:
      • Click on the print icon
      • Select File, Print
      • Press Ctrl+P
