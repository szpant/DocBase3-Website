Issuing an Extension of Time in TRAILS (7608)

Applies to
Applies to
    This procedure will outline how to issue an extension of time (EOT) in TRAILS.
1
Issuing an EOT in TRAILS
Issuing an EOT in TRAILS
    1.Log into TRAILS
    2.Use Shift Tab buttons to navigate to the Command
    3.Type IDOC select Enter
    4.Ref? Type R for registration - Type registration number
    5.Doc type? Type EOT - select Enter
    6.Tab to Man Issue Y/N Type N for No select Enter
    7.Tab to Reason? select F4 brings up List Product Action Reason
    8.Select F8 brings up second page of List Product Action Reason
    9.Type S in Sel column REGO REGIST EXT TIME ISSU select Enter
    10.Type Inspector ID number
    11.Exp/Time due - Type 16:00
    12.Exp/Due Date - Type date 6 months from expiry of last COI
    13.Due Site- Type details or select enter and the field will self populate with the current VID location
    14.EOT will be recorded in TRAILS and printed
