Dangerous Goods Vehicle Licence - Renewal (SILAS) (2750)

Note:Before printing from SILAS the "Local" default printer must be set:
  • From the Windows Start button select 'Devices and Printers'
  • Right-click on the 'trails_p1' and set as default printer

1.
Display Customer/Vehicle
Display Customer/Vehicle
    Enquiry by Customer Name or Customer Reference Number (CRN)
    Enquiry by Customer Name or Customer Reference Number (CRN)
      From the Block Menu:
      • Select P for Product and press <Enter>
      • Then select I for Industry Licence and press <Enter>
      • Then select A for Maintain Dangerous Goods Application and press <Enter>
      Accesses the Maintain Applications screen
      If Customer Reference Number (CRN) not known
      If Customer Reference Number (CRN) not known
        • Press <F7> to enter the query
        • Tab to the next field and enter:
          • Applicant's Name
          • Indicator I for Individual or O for Organisation
        • Press <F8> to execute the query

        Accesses the TRAILS Customer Search screen
        • Press <F8> again
        The customer's details with the search criteria will be displayed.

        If the customer has already been stored in SILAS, an asterisk* will appear beside the
        customer's name.

      If Customer Reference Number (CRN) is known
      If Customer Reference Number (CRN) is known
        • Press <F7> to enter the query
        • Enter the Customer Reference Number (CRN)
        • Press <F8> to execute the query
        Customer Details will be displayed

    Enquiry by Registration Number
    Enquiry by Registration Number
      From the Block Menu:
      • Select V for Vehicle and press <Enter>

      Accesses the Maintain Vehicle screen
      • Press <F7> to enter the registration number
      • Press <F8> to execute query
      • Press <ALT 3> will flow you to (Vehicle Hist) screen
      • CRN is displayed in the "Customer" field on the bottom of this screen
      • Press <F9> twice back to the Block Menu

      Follow steps above for "Enquiry by Customer or CRN" proceed to the drop down "If Customer Reference Number (CRN) is known".
2.
Lodge Application
Lodge Application
    • Press <F6> twice to go to the Applications block
    The following message appears on the Pop Up screen:-
    "There are existing products ready for renewal.
    Do you wish to create these?"

    Yes
    No
    • Highlight Yes and press <Enter>
    This will populate the Class, Expiry date and Action (RENW)
    • Press <ALT 1> to lodge the application
    • Check the date
    • Tab to OK and press <Enter>
    Note:Lodge date can be backdated
3.
Declarations
Declarations
    • Press <ALT 7>
    Accesses the Maintain Applications Declarations screen
    • Complete the declarations with Y (Yes) OR N (No)
    • Press <Control End> to save the information
    • Press <F9>
    Accesses the Maintain Applications screen
4.
Vehicles
Vehicles
    • From the Maintain Applications screen press <ALT 5>
    Accesses the Maintain Application Vehicles screen

    Note:Check that the vehicles displayed on this screen are the same as the vehicles shown on the Renewal application

    If there are no vehicles to be added or expired
    If there are no vehicles to be added or expired
      • Press <F9>
      Accesses the Maintain Applications screen

    Adding Vehicles
    Adding Vehicles
      • From the Maintain Applications screen press <ALT 5>
      Accesses the Maintain Application Vehicles screen
      • Enter the Plate number for each new vehicle on the application
      • Press <Control End> to save the information
      • Press <ALT 2> to validate the saved information
      SILAS will then obtain further vehicle information from TRAILS
      • Press <F9>
      Accesses the Maintain Applications screen
    Delete Vehicle
    Delete Vehicle
      • Highlight the vehicle that is to be deleted
      • Press <Control Delete> to delete the vehicle
      Repeat this process for each vehicle to be deleted

      When all vehicles have been deleted:
      • Press <Control End> to save the information
      • Press <F9>
      Accesses the Maintain Applications screen
5.
Fees
Fees
    To enter TRAILS receipt information
    To enter TRAILS receipt information
      • From the Maintain Applications screen press <ALT 6>
      Accesses the Maintain Application Receipts screen
      • Press <ALT 1>
      Accesses the Maintain TRAILS Receipts screen
      • Enter the TRAILS receipt information and any further relevant information in the Comment field
      • Press <Control End> to save the information
      • Press <F9>
      Accesses the Maintain Application Receipts screen

    To confirm TRAILS receipt information
    To confirm TRAILS receipt information
      In the TRAILS Receipts block of the Maintain Application Receipts screen:
      • Tab to the first blank Receipt No field
      • Press <F3>
      Accesses the SILAS TRAILS Receipts screen
      • Enter the receipt number
      • Press <Tab>
      • Highlight the relevant receipt
      • Press <Enter>
      The fields in the TRAILS Receipts block will now populate
      • Press <Control End> to save the information
      • Press <F9>
      Accesses the Maintain Applications screen
6.
Customer Relationships
Customer Relationships
    • From the Maintain Applications screen, press <ALT 0>
    Accesses Maintain Organisations screen or Maintain Individuals screen (depending on applicant being an Organisation or Individual)
    • Press <ALT 3>
    Accesses the Maintain Customer Relationships screen

    In the Relationships with Individuals Block:
    • Tab to the first blank Reln field
    • Press <F3>
    • Select DIR = Director
    • Press <Enter>
    • Enter the Customer Id number for the Director
    • Tab to the Effective field (the Effective Date automatically defaults to the Processing Date, but can be backdated if required)
    • Repeat the above process for each Director
    • Press <Control End> to save all information

    Contact Details (If known)
    Contact Details (If known)
      • From the Maintain Customer Relationships screen, press <ALT 2>
      Accesses the Maintain Contacts Details screen
      • Complete all relevant fields

      Note:<F3> can be pressed for list of values in the Device field

      • Press <Control End> to save the information
      • Press <F9>
      Accesses the Maintain Customer Relationships screen
7.
Approval
Approval
    • From the Maintain Applications screen press <ALT 2>
    • Check the date
    • Tab to OK and press <Enter>
      Note:The approve date can be backdated
8.Print Authority/Vehicle Annexure
