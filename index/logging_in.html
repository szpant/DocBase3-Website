Logging In (6100)

This procedure will outline how to log in to the Compliance Intelligence Database (CID).
1
Open Internet Explorer
Open Internet Explorer
    Select "Internet Explorer from the Start Menu, or click the icon from your taskbar.

2
Type "CID" into the address bar
Type "CID" into the address bar

3
Click "Launch Production"
Click "Launch Production"

4
Enter your login details
Enter your login details
    The user name and password are case sensitive.
    Your user name is the same as your PC login. Your password is not the same as your PC login.

    If you have forgotten your password, please submit an password reset request via the Request Portal.
    Refer: Transport Compliance > 06. Compliance Intelligence Database > Support

5
Accept the various warnings
Accept the various warnings
    Some of these messages allow you to remember the option selected.
    Select the "remember" or "do not show again" options prior to clicking anything else.







6
Ensure "Compatibility mode" is enabled (IE9 only)
Ensure "Compatibility mode" is enabled (IE9 only)
    If this is the first time accessing IntelliShare on this PC, check compatibility mode is on.

    Compatibility mode is on if this icon in the right hand side of the address bar is blue.
    If it’s not blue, click it to turn it on.

7
Successful login
Successful login
    If successful, you will be presented with the IntelliShare Homepage (or whatever module you had open last time you used IntelliShare).

8
Unsuccessful login
Unsuccessful login
    If your user name or password is incorrect after four attempts (or you do not have access to CID) you will receive this error message.

    Contact the Intel Unit for assistance via the Request Portal.
    Refer: Transport Compliance > 06. Compliance Intelligence Database > Support
