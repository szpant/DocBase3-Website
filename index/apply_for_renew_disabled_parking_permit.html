Process Application for a Disability Parking Permit (164)

Applies to
Applies to
    Checking and receipting an application for a Disability Parking Permit for:
    • a new applicant
    • invitation to apply (Permit expired)
    • adding or removing vehicles (organisation permits)
    • replacement application.
Before You Start
Before You Start
    Check Transaction Requirements

    Check Customer has:
    • Correct application form for the permit required
    • Correct documents
    • Correct payment method (if required).

    Verify applicant's address and update if required. If an agent/parent or carer has signed on the applicant's behalf, the operator must clarify whose address has been recorded prior to updating.
1
Check Documents
Check Documents
    Check that the correct sections of the application form have been completed, including:
    • The doctor, occupational therapist, or, for legally blind applications, doctor, optometrist, ophthalmologist has completed section 2
    • The duration of the medical condition is provided at Q5 (as this will determine the type of permit issued).
2
Photocopy Application
Photocopy Application
    Photocopy the front page of the application, date stamp and return to the customer for their record keeping.
3
Receipt Payment )TICA)
4
Finalise Transaction
Finalise Transaction
    Complete TMR CSC use only section of the application form:
    • CSC, receiving officer’s name, signature
    • Telephone number, Date (stamp), Receipt number
    • The receipt number must be recorded on page 3 of the application
      or
    • A copy of the document receipt is to be attached.
5
Complete checklist/scan documents
Complete checklist/scan documents
    Complete checklistComplete all items on the checklist (email template)
    Email Templates
    Scan documents/applicationScan the application and supporting documents and attach to email template.

    Note:When a customer presents an applicable concession/benefit card, staff must copy both front and back of the concession card when accepting a Disability Parking Permit application for a person who is legally blind. The copy of the card as well as the application form must then be sent to the Disability Parking Unit.
6
Forward Application
Forward Application
    • Email scanned applications and receipt/document receipt to Disability Parking (Applications Only)
7
File Documents
