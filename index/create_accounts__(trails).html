Create Accounts (TRAILS) (1430)

1. Display Customer
1. Display Customer
    • Fastpath ACCO in command line
    • Press <Enter>
      Accesses Account Maintenance AA01
    • <Tab> to [Ref]
    • Type I if individual
      P if Involved Party (Individual)
    • Enter either CRN or Involved Party (IP) Number
    • Press <Enter>
2. Create Account
2. Create Account
    • Type C in action field
      For CSC Operators
      For CSC Operators
        • <Tab> to [Operators Receipting Office]
        • <Enter> name of Receipting Office
    • <Tab> to [Type] enter account code
      If Code not Known
      If Code not Known
        • Press <F4>
          Accesses List Account Type AA08
          Navigate using <F8> and <F7> to correct account
        • Type S
        • Press <Enter>
          Returns to Account Mainenance AA01
    • Press <Enter>
      Message Displays: Create Successful

Press <F5> and repeat Step 2 if more accounts are Required
