Create an Adjustment (TRAILS) (144)

1. Fastpath ADJU
1. Fastpath ADJU
    • Type ADJU on Command Line
    • Press <Enter>
    Accesses Account Adjustment (AA07)
2. Enter Type Details
2. Enter Type Details
    • Type C in action field
    • <Tab> to [Type?] enter Type Code TRFCUST
    If code Not Known
    If code Not Known
      • Press F4 in [Type? Origin]
      Accesses List Adjustment Types (AA09)
      • Press F7 or F8 to navigate
      • <Tab> to required “Type” Type S
      • Press <Enter>
      Returns to List Adjustment Types AA09
3. Enter Adjustment Details
3. Enter Adjustment Details
    • Enter Amount to be adjusted [Control Total]
    • Enter Account Numbers in [Account No]
      If Account Numbers not Known Search for Account Numbers (TICA)
    • Enter Debit (DR) or Credit (CR) in [DR/CR]
    • Enter Dollar Totals in [Amount]
    • Type S beside each entry
    • Press <Enter>
4. Enter Adjustment Details Replacing Main Roads Govn Plates
4. Enter Adjustment Details Replacing Main Roads Govn Plates
    Replacing Main Roads government plates with standard plates
    Note:This account adjustment is processed after the plate replacement has been processed in TRAILS.
    • Enter Amount to be adjusted in the Control Total [Replacement Plate fee Total]
    • <Tab> to [Sel]
    • 'S' Select
    • Enter Account Number - 602438299 DR then plate fee amount
    • <Tab> to [Sel]
    • 'S' Select
    • Enter Account Number - 600135755 CR then plate fee amount
    • Press <Enter>
5. Enter Text Message (If Required)
5. Enter Text Message (If Required)
    Note:Some adjustments will automatically access the Text Editor
    • Type X in action field (if required)
    Accesses Text Editor SM11
    • Type U in aciton Field
    • Enter text message
    • Press <Enter>
    Message Displays: Update Successful
    • Press <F12>
    Returns to Account Adjustment AA07
6. Enter Text Message Replacing Main Roads Govn Plates
6. Enter Text Message Replacing Main Roads Govn Plates
    Note:Some adjustments will automatically access the Text Editor
    • Type X in action field (if required)
    Accesses Text Editor SM11
    • Type U in aciton Field
    • Enter text message
      'As per letter of authority Government Plate no.XXXXXX has been replaced with XXXXXX'
    • Press <Enter>
    Message Displays: Update Successful
    • Press <F12>
    Returns to Account Adjustment AA07
7. Update Adjustment
7. Update Adjustment
    • Press <Home> then Type C
    • Press <Enter>
    • Press <Home> and type U in Action Field
    • Enter text message press <Enter>
    • Press <F12>
    Message Displayed "Create Successful"
