Conducting a Search (7015)

This procedure will outline how to search for an entity in the CID.
1
Open Find and Explore module
Open Find and Explore module
2
Enter search term
Enter search term
    Type the search word/CID ID/ACN/CRN into the search box.

    You can search on all fields in the database via this one search box

    When searching for organisations by name it is best practice to use a short version of the company name. e.g. Don Watson Equipment Hire Pty Ltd - search should be Don Watson - this will give you a complete return of the organisations.

    When searching for a person it is best practice to use the first name and surname only, and then use additional filtering if large results have been returned. (See 7. Additional filtering).

3
Filter records by entity type
Filter records by entity type
    By default you are presented with all entity types that match your search term. The Summary pane displays the number of records found matching your search term and breaks them up into the various entities.

    You can filter the results to only show a particular entity type by selecting a entity in the Summary pane. Records are then displayed in a more tabular layout.

4
Score indicator
Score indicator
    In the Details pane a Score indicator is displayed.

    This shows how closely a match the record is to your search term.

    A full Score indicator is an exact match. A partially filled Score indicator shows "fuzzy" matches.



    Example
    Example
      A search for a Case with a CID ID of "CAS12" will also return results of "CAS121", "CAS122", and "CAS21". This is due to the "fuzzy" search capabilities.
5
Expand the entity to view linked entities
Expand the entity to view linked entities
    You can view entities linked to a record by clicking the expand items icon (for example, to view the Cases linked to a company).


6
View record data using Central pane and Details pane
View record data using Central pane and Details pane
    Select a record in the Central pane to view the record data in the Details pane.



    If the selected record is a linked entity, the Details pane displays buttons at the top of the data fields.
    Click these buttons to view the record details for the main entity, details of the link, and the the connected entity.

    The blue highlighting on the button indicates what record the details displayed relates to (in the image below, the link details are being displayed).


7
Additional filtering
Additional filtering
    When you have conducted a search and have selected an entity type in the Summary pane, you are able to apply additional filters to reduce the records.

    Click on the filter icon and enter the term to filter on, and click "Set".

    Example: filter a list of Cases to only show the cases where the "Category" is "PT".
    Example: filter a list of Cases to only show the cases where the "Category" is "PT".


    When a filter is active the filter Icon displays blue, and a filter icon appears in the heading of the Central pane.



    To remove the filter, click the filter icon, delete the term, and click "Set".
