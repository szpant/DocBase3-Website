Access ROAMS, Add/Edit Interception Sites and Raise Work Orders (7520)

This procedure will outline how to Access ROAMS, Add or Edit Interception Sites and raise Work Orders.
1
Access ROAMS
Access ROAMS
    1
    		Go to TSDM website
    https://www.tap.qdot.qld.gov.au/tsdm/html/tsdm-index.html
    2
    		Select MyTSDM
    3
    		Enter user name and password - select Log in
    4
    		Select the link for ROAMS to open the web application
    5
    		Select:
    - Sites to update or add an interception site or
    - Work Orders to create a work order.
    6
    		When you have finished in ROAMS you must sign out
    7
    		Select the arrow on the MYTSDM toolbar to sign out
    8
    		For assistance with ROAMS email tsdm@tmr.qld.gov.au
2
Add a Site
Add a Site
    1
    		Log into ROAMS
    2
    		Select Sites
    3
    		Select add
    4
    		Select the area from the drop down menu
    5
    		Select Heavy Vehicle Interception Sites from site type
    6
    Enter details
    Enter details
      Enter details in the following fields:-
      • Description for example. Dockers Creek West Bound 3km East of Jondaryan. Ensure the description you enter is clear and succinct.
      • Longitude and Latitude. The co-ordinate must be in decimal degrees. If using Google Earth Pro, the settings can be changed in Tools - Options
      • Type in the Site Name for example, Dockers Creek
      • Add any Comments for example. Mobile intercepts only, site not safe for static intercepts.
      • Add a link to the SharePoint Interception database in SharePoint Documents. Copy the URL below for the interception site page in Compliance SharePoint, right click in the text box, select add link, paste link. https://intranet.tmr.qld.gov.au/sites/csb-reg/Pages/ROAMSInfo.as
      • To access the SharePoint Site from ROAMS, Control key and click on the link.
    Complete the following
    Complete the following
      Complete the Following Fields:-
      • Location of WIM and or ANPR if relevant
      • Frequency Level, for example. Daily, Weekly, Monthly, Seasonal use only
      • Direction of travel
      • If the site is Not Operational, mark the check box and enter text in reason why
      • Length of the speed zone in metres, between the 2 speed signs if the speed limit is reduced while the site is in operation. (Refer attachment 1 below)


      • Speed limit for the site when the site is not in operation
      • Operational speed limit while the site is in operation
      • Site Surface from the drop down menu
      • Site Length in metres. (refer attachment 2 below)


      • Concrete Slab Length in metres if applicable
      • Concrete Slab Width in metres if applicable
      • Site Width in metres. (Refer attachment 3 below)



      • Separation from Carriageway from the drop down menu
      • Region Name?
      • Fields from District Number to Locality inclusive will self-populate
      • Mark the Site Safety Plan check box (all static interceptions sites must have a Site Safety Plan)
      • Select Category 1,2 or 3 from the Site Category drop down menu. Refer to Interception Sites (7490) Insert link http://intranet.qdot.qld.gov.au/tmr/docbase3.nsf/index/interception_sites
      • If the site is Inactive, select the reason from the Inactive Reason drop down menu and add text in the Comments section
      • Select Update when all appropriate fields have been completed

    For any ROAMS issues contact Wayne Dale or email tsdm@tmr.qld.gov.au
3
Complete the data entry form
Complete the data entry form
    If any details about an interception site change ROAMS must be updated
    1
    		Log into ROAMS
    2
    		Select Search
    3
    		Select Search by Area and Site Type
    4
    		Select the relevant Area
    5
    		Select Heavy Vehicle Interception Sites - click OK
    6
    		Select the relevant site from the list - click OK
    7
    		Update relevant details
    8
    		Select Update
    9
    		Send an email to the STI advising of changes
4
Save the record
Save the record
    1
    		Log into ROAMS
    2
    		Select Work Orders
    3
    		Select Create a new Work Order
    4
    		Select the relevant Area from the drop down menu
    5
    		Select the appropriate Work Order Type from the drop down menu
    6
    		Complete the Reported by
    7
    Work Order Description
    Work Order Description
      Work Order Description must include Site ID and Name.
      Description of work required, for example 80 km/h speed limit sign damaged requires replacing or large pot hole at northern end of site.
      Include text advising photos have been added to the Interception Site page
    8
    		Select Submit
    9
    		Upload photos to the Interception Site page.
    10
    		Log out of ROAMS when finished
