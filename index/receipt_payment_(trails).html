Receipt Payment (TRAILS) (454)

1. Access Payment Screen (if not already accessed)
1. Access Payment Screen (if not already accessed)
    • Fastpath PAYM in Command Line
    • Press <Enter>
    Accesses Payment (AP01)
    • <Tab> to [Ref?] type C and CRN
    • Press <Enter>
2. Select Product or Pay to Account
2. Select Product or Pay to Account
    Select Product
    Select Product
      • <TAB> to [Sel]
      • Type S on [Product/Offence Num and Description?]
    or
    Pay to Account (If required)
    Pay to Account (If required)
      • Press <Home> then Type P
      • <TAB> to [Pay to Account]
      • Enter amount to be paid
3. Pay using relevant Pay Method
3. Pay using relevant Pay Method
    CASH
    CASH
      • <Tab> to [Cash]
      • Type in amount recieved from customer
      • Press <Enter> (Receipt will automatically print)
    EFTPOS
    EFTPOS
      • <TAB> to [Method?] below [Cash]
      • Type EFTPOS in [Method?]
      • Enter amount received
      • Press <Enter> (Receipt will automatically print)
    CREDIT CARD
    CREDIT CARD
      • <TAB> to [Method?] below [Cash]
      • Type the relevant code in [Method?]:
      • MCAR (Master Card) or
      • VISA

      Enter the following details
      Enter the following details
        • Amount recieved in [Amount]
        • Last 4 digits of Credit Card Number in [Reference Number]
        • Surname on Credit Card in [Drawer]

      • Press <Enter> (Receipt will automatically print)

        Note:Credit Card details must not be written down when provided verbally, and must be blacked out when provided in writing.
    CHEQUE
    CHEQUE
      • <Tab> to [Method?] below [Cash]
      • Press Press F4
      Accesses List Payment Item Methods (AP11)
      • Type S beside Cheque type
      • Press <Enter>
      Accesses Payment (AP01)

      Enter the following details
      Enter the following details
        • Amount recieved in [Amount]
        • Number in [Reference Number]
        • Cheque Drawer name in [Drawer]
        • Bank initials/branch in [Bank/Branch]

      • Receipt will automatically print if not required:
      • Select N for [Print Receipt Y/N]
      • Press <Enter>
    Pay From Account (If Applicable)
    Pay From Account (If Applicable)
      • <Tab> to [Pay From Account]
      • Type amount being deducted in [Pay From Account]
      • <Tab> to [Print Receipt (Y/N)]
      • Type N in [Print Receipt (Y/N)]
      • Press <Enter> (Receipt will NOT print)
4. Reprint Receipt (if required)
4. Reprint Receipt (if required)
    • Press <Home>
    • Type I
    • Press <Enter>
    Can only reprint before exiting the payment screen
5. Record Text if Pay to Account
5. Record Text if Pay to Account
    • Type X in Action Field
    • Press <Enter>
    Accesses Text Editor
    • Type U in Action field
    • Enter reason & details for payment to account -including CRN
    • Press <Enter>
    Message displays: Update SUCCESSFUL
    Print text on back of receipt for non-Queensland customers.
