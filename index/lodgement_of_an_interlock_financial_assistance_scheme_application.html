New Application
New Application
    Applies to
    Applies to
      Accepting, receipting and forwarding an application for an Alcohol Ignition Interlock Financial Assistance Scheme (IFAS) application.
    Before you Start
    Before you Start
      • Identify customer for whom application is being lodged (Check CRN in TICA)
      • Check customer will be or is required to comply with the Qld AIIP if:
        • Qld licence holder either check if they have a Qld licence with an "I" condition licence code or use AIIP verification process in TICA
        • Non-Qld licence holder check that their Non-Qld licence is subject to an "I" condition
      • Confirm applicant has application form and required supporting documentation

      Refer: Driver Licence > 3. Enforcement > Alcohol Ignition Interlock Program

    1
    Check Documents
    Check Documents
      Application Form
      • Check the Application Form has been completed correctly
      Supporting Documents
      • Check that all supporting documents have been provided and if any payment card details (credit card or bank card) exist, photocopy then redact (with permanent marker) prior to scanning

      Refer: Driver Licence > 3. Enforcement > Alcohol Ignition Interlock Program > Financial Assistance Scheme > Application Requirements
