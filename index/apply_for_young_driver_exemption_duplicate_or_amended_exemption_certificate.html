Apply for Young Driver Exemption/Duplicate or Amended Exemption Certificate (3171)

New Application
New Application
    Applies to
    Applies to
      Accepting, receipting and forwarding an application for an exemption from the following young driver restrictions or requirements:
      • High Powered Vehicle - applies to P1/P2 Licence holders
      • Late Night Driving - applies to P1/P2 Licence holders
      • Logbook Requirements - applies to Learner Licence
    Before you Start
    Before you Start
      • Check that the customer has the correct payment method
      • Identify customer for whom exemption is being lodged (Check CRN in TICA)
      • Confirm applicant has application form and required supporting documentation and holds eligible licence type
    1
    Check Documents
    Check Documents
      Application Form
      • Check the Application Form has been completed correctly
      Supporting Documents
      • Check the Supporting Statement and any required documents are completed
    2
    		Lodge Application (TICA)
    3
    Forward Application
    Forward Application
      Internal applications:
      • Write the receipt number on the application
      • Scan and email all pages daily to CPU (Exemptions)
      • Subject for email should be SDRE/LNDR/HPVE:<Surname><CRN>

      External applications:
      • Advise customer to post application and any supporting documentation to CPU (Exemptions) PO Box
    4
    		File Lodged Application
Duplicate/Amended Certificate
Duplicate/Amended Certificate
    Applies to
    Applies to
      Accepting and forwarding a request for:
      • A duplicate Certificate of Exemption
      • An amendment to a Certificate of Exemption
    Before you Start
    Before you Start
      • Confirm applicant has a written request and any required supporting documentation
    1
    Forward Request
    Forward Request
      Internal applications:
      • Scan and email all pages to CPU (Exemptions)
      • Subject for email should be "YDEC - <Duplicate/Amended> - <Surname> <CRN>"

      External applications:
      • Advise customer to forward written request and any supporting documentation to CPU (Exemptions)
