ANPR Entity
ANPR Entity
    This procedure will outline how to create an ANPR Entity in CID.

    Limited Staff PA(I)'s, STI's, FIO's and Heavy Vehicle Analysis have the ability to create this entity type in CID.

    Should you not have access to add the ANPR Entity, you should email compliance.intelligence@tmr.qld.gov.au with the ANPR notice/document requesting a record to be created.

    Note:ANPR and WiM documents should be sent together.
