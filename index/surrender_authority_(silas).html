Surrender Authority (SILAS) (2871)

1.
Display Customer
Display Customer
    From the Block Menu:
    • Select P for Product
    • Select E for Authority Enforcements
    Accesses the Maintain Authority Enforcements screen

    If Customer Reference Number (CRN) not known
    If Customer Reference Number (CRN) not known
      • Press <F7> to enter the query
      • Tab to the next field and enter:
        • Applicant's Name
        • Indicator I for Individual or O for Organisation
      • Press <F8> to execute the query
      Accesses the TRAILS Customer Search screen
      • Press <F8> again
      The customer's details with the search criteria will be displayed.

      If the customer has already been stored in SILAS, an asterisk* will appear beside the
      customer's name.

      If an asterisk does not appear beside the customer's name:
      • Press <ALT 4>
      The customer ID is now stored in SILAS and an asterisk now appears beside the customer's
      name
      • Highlight the relevant customer and press <ALT 1> to select
      The customer details are then brought back to the Maintain Applications screen.

    If Customer Reference Number (CRN) is known:
    If Customer Reference Number (CRN) is known:
      • Press <F7> to enter the query
      • Enter the Customer Reference Number (CRN)
      • Press <F8> to execute the query
      Customer Details will be displayed
2.
Surrender Authority
Surrender Authority
    If more than one product shows, Tab to the Application/Products block and arrow down to the required product
    • Press <ALT 5>
    The Enforcement Action dialog box will appear on the screen

    Note: If more than one product shows
    • Check the date
    • If the date needs to be changed, Tab to Effective Date and enter the correct date
    • Tab to OK and press <Enter>
    This will surrender the authority and expire all vehicles attached to the authority
