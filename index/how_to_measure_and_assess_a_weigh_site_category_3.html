How To: Measure and Assess a Weigh Site Category 3 (7583)


This procedure will outline how to measure a weigh site to ensure it meets the minimum specifications for a category 3 weigh site
1
Longitudinal Plane Gradient
Longitudinal Plane Gradient
    Measure the length of the area where the vehicle is to be weighed (Weighing Area), along with the approach and departure wheel paths, (Refer: Weigh site requirements section).
    • Mark the ground to indicate the limits of each area.
    • Anchor one end of the string line on a portable scale or witches hat at one end of the area being measured.
    • Place the line level in the centre of the string and pull the string tight.
    • Slowly raise the free end of the string off the ground until the line level indicates that the string is level.
    • Measure the height of the string from the ground and record the measurement in official notebook.
    • Check the measured height difference does not exceed:
      • Weighing area 5% up to 250mm
      • Approach wheel path 5% up to 700 mm
      • Departure wheel path 5% 700 mm
2
Transverse Plane Gradient
Transverse Plane Gradient
    Measure the width of the area where the vehicle is to be weighed (Weighing Area), along with the approach and departure wheel paths, (Refer: Weigh site requirements section).
    • Mark the ground to indicate the limits of each area.
    • Anchor one end of the string line on a portable scale or witches hat at one end of the area being measured.
    • Place the line level in the centre of the string and pull the string tight.
    • Slowly raise the free end of the string off the ground until the line level indicates that the string is level.
    • Measure the height of the string from the ground and record the measurement in official notebook.
    • Check the measured height difference does not exceed:
      • Weighing area 175 mm.
      • Approach wheel path can be more than 175 mm or 5%
      • Departure wheel path can be more than 175 mm
3
Weigh Site Unevenness Over a 5m Span
Weigh Site Unevenness Over a 5m Span
    Any irregularities that are in the wheel path areas must not be deeper than the maximum limits.
    To measure uneven surfaces in the wheel path:
    • Anchor one end of the string line at one end of the wheel path area being measured.
    • Place the other end of the string line at the end of the wheel path area at ground level and pull the string line tight.
    • Measure the depth of any uneven surfaces and record the measurement in official notebook.
    • Check the measured depths do not exceed:
      • Weighing area 50 mm.
      • Approach wheel path 100 mm
      • Departure wheel path 150 mm
4
Documentation
Documentation
    When the interception site has been assessed, the Transport Inspectors that measured the site are required to provide statements verifying the site was measured and meet the criteria for a Cat 3 weigh site. This will negate the requirement to measure the site every time the site is used.

    Evidence of the site assessment should be captured in DMS and the Interception Site SharePoint site.
    Evidence includes:
    • photographs or video of the site assessment
    • drawings, plans and note book entries

    ROAMS, the site safety plan and the SOP must also be updated.
