Code of Practice – Light Vehicles
Code of Practice – Light Vehicles
    The Code of Practice – Light Vehicles was replaced by the National Code of Practice for Light Vehicle Construction and Modification on 1 November 2012.

    To ensure vehicle owners who were working on modifications when the change over occurred, this code was continued in use until 30 April 2013. During this period, modifications were recorded on the modification plate and certificate of modifications with a Q in front of the code (e.g. LA1 was recorded as QLA1).
