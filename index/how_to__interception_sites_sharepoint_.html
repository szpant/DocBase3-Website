Interception Sites SharePoint Procedures (7522)

This procedure will outline how to add Photographs and Documents to the Interception Site Compliance SharePoint then edit them if required.

Senior Transport Inspectors, Compliance Administration Officers and Compliance Managers have update access to the Interception Site document library. All visitors have Read access.
1
Add an Interception Site
Add an Interception Site
    1
    		Interception Lookup List
    2
    		Select new item
    3
    		Type the name of the Interception Site in Title
    4
    		Select the Region from the drop down menu
    5
    		Select Save

2
Add Documents
Add Documents
    1
    		Interception Site Information
    2
    		Select Upload
    3
    		Select Browse in Choose a file, select the file you wish to save
    5
    		Add Version and date of file in Version Comments, select OK
    6
    		Select Quick Edit here, the saved file will appear in the table.
    7
    Enter details
    Enter details
      • Region from the drop down menu
      • ROAMS Site ID
      • Category from the drop down menu
      • New documents remain Current
      • Site from the drop down menu
    8
    		When finished exit the table and the document will update.

3
Add Photographs
Add Photographs
    1
    		Interception Site Images
    2
    		Select Upload
    3
    		Select Browse in Choose a file, select the photo you wish to up load
    4
    		Add Version and date of file in Version comments, select OK
    5
    		Select 'Quick Edit' here, the saved photo will appear in the table
    6
    Enter details
    Enter details
      • Region from the drop down menu
      • ROAMS Site ID
      • Category from the drop down menu
      • New photos will remain Current
      • Site from the drop down menu
      • File will update and save when you return to Home Page
    Rename Photograph
    Rename Photograph
      • If the photo requires renaming select the 3 dots, select 3 dots again, select Rename. Type in the fill Name using the required naming convention.
      • Site from the drop down menu
      • Region from the drop down menu
      • Image from Category drop down menu
      • New photos will remain Current
      • ROAMS Site ID
      • Enter Comments if required
      • Select Save

4
Replacing Photographs and Documents
Replacing Photographs and Documents
    If you are replacing a document such as a SOP or Photograph with the same name, it will be saved as the latest version.

    If a new name is used, please update the properties of the new document (the default category is "New document") and remove the Current flag on the old document.

    1
    		Select the 3 dots beside the document/photo name, select the 3 dots again, select Rename
    2
    		Un-check the Current check box
    3
    		Add comments
    4
    		Select Save
