Formal Warning Entity (7437)

This procedure will outline how to create a Formal Warning entity in CID.

The Formal Warning Entity has been created to enable officers to record Formal Warnings issued to Organisation / Persons.

All staff have the ability to create this entity type via CID.
1
Select Data Entry
Select Data Entry
2
Select Formal Warning from the Form Menu
Select Formal Warning from the Form Menu
3
The Formal Warning Form will be displayed
The Formal Warning Form will be displayed

Entity - Base Details - Below is a description of each field and how it should be utilised.
1
Record ID
Record ID
    • System generated, cannot be changed.
2
Name / Title
Name / Title
    Mandatory field
    • The name of the person or organisation that the formal warning was issued to
3
Customer Reference Number
Customer Reference Number
    Mandatory field
    • The customer reference number as found in TICA must be recorded
4
Vehicle Registration Number
Vehicle Registration Number
    • The registration number of the vehicle involved
    • Use the radio buttons to indicate the involvement of either used vehicle or permitted the use of the vehicle.
5
Offence Code
Offence Code
    • The code for the offence that has been committed
6
Formal Warning Notice Number
Formal Warning Notice Number
    Mandatory Field
    • The Notice Number of the Formal Warning issued.
7
Issue Date
Issue Date
    Mandatory Field
    • Self explanatory
8
Issued Time
Issued Time
    • Self Explanatory
9
Withdrawn Date
Withdrawn Date
    • The withdrawn date is to be recorded if applicable
    • If a withdrawn date is recorded the status must be changed to Withdrawn
10
Issuing Region
Issuing Region
    Mandatory field
    • Self Explanatory
11
Issued by
Issued by
    Mandatory Field
    • Self Explanatory
12
Status
Status
    Mandatory field
    • Current
    • Withdrawn - If the status is withdrawn a withdrawn date must be entered.
13
Notes
Notes
    • This is a free text field, where officers may wish to record any relevant details relating to the Formal Warning.
