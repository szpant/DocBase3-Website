Demonstrating Gearbox Skills (631)

Transmission Types
Automatic
Automatic
    An automatic transmission may be taken to be a transmission system on a motor vehicle in which gear changing is operated automatically in accordance with the rate of movement of the vehicle ("car speed") or rate of the revolutions per minute of the engine ("engine speed") rather than manually by the driver.

    Criteria for Determining if Automatic
    Criteria for Determining if Automatic
      A motor vehicle may be taken to have an automatic transmission if any of the following applies:
      • There is no driver-controlled system for disconnecting the drive between the engine and the driving wheels other than the system for selecting gear ratios
      • Gear changing occurs automatically in accordance with car or engine speed rather than manually by the driver
      • Gears are changed manually by using buttons on the steering wheel or dashboard or with the gear lever or gearshift, but then the system automatically operates the clutch, selects the gears, engages the gears and regulates the engine speed during gear changing
      • The driver selects the automatic mode of gearbox by pushing or pulling a device (button or lever) and then each gear change and decision about the gear to be engaged are controlled by the gearbox.

        The automatic transmission may also have a clutch that is operated by a pedal or a lever         but is used only for starting and stopping.

    Types of Automatic Transmissions
    Types of Automatic Transmissions
      Cars Tiptronic, Sequentronic, Selespeed
      What is Tiptronic/Selespeed Transmission
      What is Tiptronic/Selespeed Transmission
        If a vehicle has Tiptronic/Selespeed or similar type transmission fitted, it can be driven in either automatic or manual mode. These type of vehicles do not have a manual clutch. In the manual mode some vehicle manufactures use buttons on the steering wheel to select the next gear, while others use a lever in the dashboard or in the centre consul.
      Heavy Vehicles Eurotronic II systems

Manual
Manual
    A manual transmission may be taken to be transmission system on a motor vehicle that required the driver to manually change gears on every occasion by disengaging and engaging the drive between the engine and the driving wheels by using a clutch pedal and a gear lever or gearshift.
Non-synchromesh
Non-synchromesh
    A non-synchromesh transmission (commonly known as a 'crash box') may be taken to be transmission system on a motor vehicle that requires the driver, when changing gears, to manually synchronise the speed of the driving and driven gears with the engine speed by using the clutch pedal and the gear lever/gearshift before engaging the drive between the engine and the driving wheels. All forward gears must be non-synchromesh.
Synchromesh
Synchromesh
    A synchromesh transmission may be taken to be transmission system on a motor vehicle that has a system of clutches that automatically synchronises the speeds of the driving and driven members before engagement to avoid shock in gear changing and to reduce noise and wear. However, it is acceptable for first gear in this type of transmission to be non-synchromesh.

Licence Issuing
Classes C and LR - Automatic and Manual
Classes C and LR - Automatic and Manual
    If a licence holder passes a practical test for classes C or LR in a vehicle with a manual transmission they are issued a manual class C or LR licence.

    If a licence holder passes a practical test for classes C or LR in an automatic vehicle or vehicle fitted with Tiptronic / Selespeed or similar type transmission, the code A for "automatic only" must be imposed on the licence.

    What is Tiptronic/Selespeed Transmission
    What is Tiptronic/Selespeed Transmission
      If a vehicle has Tiptronic/Selespeed or similar type transmission fitted, it can be driven in either automatic or manual mode. These type of vehicles do not have a manual clutch. In the manual mode some vehicle manufactures use buttons on the steering wheel to select the next gear, while others use a lever in the dashboard or in the centre consul.


    If a class C (manual) licence holder passes a practical test in an automatic LR class vehicle, the demonstration of manual gearbox skills in the car is recognised for the issue of a manual LR licence.
Classes MR, HR, HC and MC - Synchromesh and non-synchromesh
Classes MR, HR, HC and MC - Synchromesh and non-synchromesh
    A non-synchromesh gearbox is one in which all gears are non-synchromesh (eg. Roadranger). The following are not eligible for recognition of non-synchromesh gearbox skills:
    • Test vehicle that has a gearbox with first gear only non-synchromesh
    • A test in which the applicant double declutches using a synchromesh gearbox.

    All class MR, HR, HC & MC applicants who have only ever passed a practical test in a heavy vehicle (over 4.5 tonne GVM) with a synchromesh gearbox, will have a B condition endorsed on their licence. This means they can only drive heavy vehicles with a synchromesh or automatic gearbox.

    If a class MR, HR, HC unrestricted (e.g non-synchromesh) licence holder passes a practical test for a higher class (e.g class MR to class HR/HC or class HC to class MC) in a vehicle with a synchromesh gearbox, the new licence class is also unrestricted.

    This is because TMR recognises they have already demonstrated their non-synchromesh gearbox skills in a heavy vehicle.

    If a class C (manual) licence holder passes a practical test for class MR or HR in an automatic vehicle, the MR/HR licence must be endorsed with a synchromesh (B) condition, not an automatic (A) condition.

    If a class CA (automatic) licence holder passes a practical test for a class MR or HR in an automatic vehicle, the MR/HR licence must be endorsed with an automatic (A) condition.

    If a class MR, HR, HC, MC licence holder with an (A) condition passes a practical test in a manual car. The (A) condition is removed to allow them to drive a class C or LR manual vehicle but a synchromesh (B) condition must be added to the licence which applies to the MR, HR, HC, MC licence.

    Removing the (B) Condition
    Removing the (B) Condition
      To remove a (B) condition from a class MR,HR,HC, MC licence the person must demonstrate non-synchromesh gearbox skills by passing a practical test in a heavy vehicle (class MR or above) with a non-synchromesh gearbox.
