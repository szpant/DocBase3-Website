Create Involved Party - Individual (TRAILS) (1241)

1. Display Involved Party
1. Display Involved Party
    • Fastpath LINP in command Line
    • Press <Enter>
    Accesses Involved Party Search IP28
    • <Tab> to name, enter party's Surname, First initial
    • Press <Enter>
    • Type M in action field
    • Press <Enter>
    Accesses Involved Party Maintenance IP29
2. Complete Involved Party Details
2. Complete Involved Party Details
    • Enter the Following Details
      [Given Names] - First Name, Middle Name
      [IP Type] - I
      [Origin] - Enter Code
      If code not known
      If code not known
        • Press <F4>
        Accesses Involved Party Origin PV10
        • <Tab> to correct origin type S
        • Press <Enter>
        Returns to Involved Party Mainenance IP29
      [Qual Type] - Enter Code
      If code not known
      If code not known
        • Press <F4>
        Accesses Involved Party Qualification Type List SS35
        • Navigate using <F8> and <F7>
        • <Tab> to correct Qual type S
        • Press <Enter>
        Returns to Involved Party Maintenance IP29
      Qualification Types
      [Qual Ref Num] - if Required
      [Qual Effect Dt] - Enter date Qual is to start

      Contact Address
      [Property Id] - if applicable
      [Street Number]
      [Street Name]
      [Sub/Town]
      [Phone Num]
      [Fax Number] - if applicable
      [Customer Num] - Record Involved Party's CRN

      Note:If customer does not have a CRN, the customer is to be created in TRAILS and the CRN recorded in this field

      Individual Details
      [Employee Num]
      [Job Class] - eg AO4
      [Emp Basis] - F= Fulltime/ P= Parttime / C= Casual
    • Press <Enter>
    Message Displays: Add Successful
3. Link Employee to Location (if required)
3. Link Employee to Location (if required)
    • Type P in Action field
    • Press <Enter>
    Accesses Involved Party Employer Maintenance IP36
    Display Employer
    Display Employer
      • Type E in Action Field
      • <Tab> to [REf Num]
      • Type Q enter Name of Location
      • Press <Enter>
    Complete Employment Role Details
    Complete Employment Role Details
      • Press <Home> type A in Action Field
      • <Tab> to [Qual Type] enter code
      If code not known
      If code not known
        • Press <F4>
        Accesses Involved Party Qualification Type List SS35
        • Navigate using <F8> and <F7>
        • <Tab> to correct Qual type S
        • Press <Enter>
        • Returns to Involved Party Maintenance IP29
      • <Tab> to [Primary Lication(Y/N)] type Y or N
      • Enter Start date at [Effective Date]
      • Enter Finish Date at [Expiry Date] (if applicable)
      • Press <Enter>
      Message Displays: Add Successful
4. Record User Id (if required)
4. Record User Id (if required)
    • Press <F12>
    Returns to Involved Party Maintenance IP29
    • Type R in Action Field
    • Press <Enter>
    Accesses Maintain User Profiles SS06
    • Press <Home>
    • Type S in Action Field
    Accesses Assign Involved Party Userid
    • Type A in Action Field
    • <Tab> to [Userid]
    • Enter RACF issued Userid
    • Press <Enter>
    Message Displays: Userid Assigned Successfully
