EOI - Name in Full (2825)

Initials/No Name on Documents
Category A
Category A
    Category A documents with no name or initials only showing for the first and/or middle names are unacceptable. A further category A document must be supplied showing the full name.

    Only exception - Interstate Licence on NEVDIS
    Only exception - Interstate Licence on NEVDIS
      If an interstate licence is produced showing a middle initial only it may be accepted provided NEVDIS shows the full name for the driver licence presented.
      The full name must be recorded on the Qld driver licence.

      Note:No other alias or previous names attached to that driver licence record on NEVDIS can be used.
Category B
Category B
    Category B documents showing initials and surname are acceptable providing the initials and full surname match the full name on the Category A document. If the full surname is not shown on the Category B document then it should be accompanied by documentation on letterhead from the issuing institution which links the abbreviated surname to the full surname.

    Financial institution debit/credit card
    Financial institution debit/credit card
      If the holders name is not embossed/printed on the debit/credit card the document may be accepted if supported by a letter/ printed transaction update on an original financial institution letterhead confirming the holders:

      • Full name
      • Address
      • Signature
      • Account number &/or card number

      Note:If the financial institution will not provide the letter, the Customer Identity Declaration F2031 may be used instead.

Sequence of First & Middle NamesThe sequence of the first and middle name/s on the EOI documents does not need to be the same. As long as the same names appear on all documents they may be accepted. However, the TMR product (eg. Driver Licence) is to be issued in the name as it appears on the Category A document when the name sequence is clearly identifiable from that document.

If the name order is not clear from the documentation provided, seek clarification from the customer and if the name sequence is still unclear, officers may request further documentation from the customer to clarify.

Foreign names with accents or diacritics
Foreign names with accents or diacritics
    Accents and diacritics are marks added to letters in foreign languages which sometimes make them into different letters when transliterated into English. If a customer's name in their Category A document has a diacritic/accent which can create a different combination of letters in English, then the customer may choose to have this alternative spelling on their driver licence. This only applies to German and the Scandinavian languages (Danish, Finnish, Norwegian and Swedish) since diacritics in other foreign languages can usually not be accurately transliterated into English. Identifying Family NameIf the family name is not clearly identifiable it is to be taken as the last name appearing on the Category A document, when it is read from left to right.

An exception to this rule is an Australian Visa as the Family Name is always displayed first when reading left to right on this document.

Note:Name sequence (ie. the order of the given, middle and family names) on official documents issued from other countries may not be the same as issued within Australia.

If the name order is not clear from the documentation provided, seek clarification from the customer and if the name sequence is still unclear, officers may request further documentation from the customer to clarify.

Child Surname on Australian Birth Certificates
Child Surname on Australian Birth Certificates
    Prior to 1 July 1974. Australian Birth Certificates only showed the child's given/Christian names in the "CHILD" field.

    In this situation the child traditionally assumed the father's surname/ family name if the father's surname/family name is registered on the birth ertificate.

    From 1 July 1974 Australian Birth Certificates recorded the Child's surname/family name in the 'CHILD' field together with the child's given/Christian names.

    The last name in this field is identified as being the child's surname/family name.
    However in some instances:
    • A Birth certificate does not show the father's surname/
      family name
      or
    • The Father's surname/family name is different to the mother's surname/ family name (eg parents were not married)
    In these situations the surname/family is determined by:
