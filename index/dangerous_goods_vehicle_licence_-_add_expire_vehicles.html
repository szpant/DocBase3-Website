Dangerous Goods Vehicle Licence - Add/Expire Vehicles (2751)

Note:Before printing from SILAS the "Local" default printer must be set:
  • From the Windows Start button select 'Devices and Printers'
  • Right-click on the 'trails_p1' and set as default printer


1.
Update Vehicle Details
Update Vehicle Details
    • Select P for Product
    • Select I for Industry Licence
    • Select M for Maintain Industry Licenses
    Accesses the Maintain Industry Licenses screen
    • Press <F7> to enter the query
    • Enter the Customer Reference Number (CRN)
    • Press <F8> to execute the query
    The Industry Licence Authority and associated Vehicles will display
    Add / Replace Vehicle
    Add / Replace Vehicle
      • Press <F6> twice
      • Arrow to the first blank Vehicle field and enter the Plate Number
      • Tab to Effective Date field and enter the Effective Date (if different to today's date)
      • Press <Control End> to save the information
      Repeat the above process if there are several vehicles to be added

      Note:A dialogue box will appear to prompt the operator to ensure the vehicle/s have a current inspection and satisfy the requirements.

      If the vehicle/s satisfy the requirements, press <F9> to continue processing.

    Expire Vehicle
    Expire Vehicle
      • Highlight the vehicle that is to be expired
      • Enter the Expiry date
      • Press <Control End> to save the information

      If there are several vehicles to be expired:
      • Enter the expiry dates
      • Press <Control End> to save all the information at the same time
