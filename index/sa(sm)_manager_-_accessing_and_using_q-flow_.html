SA(SM)/Manager - Accessing and Using Q-Flow (7357)


When to log inSA(SM)/Managers should log in to Q-Flow every morning before the centre opens and check all rostered operators for that day have the correct unit access.
Refer below: Getting Started > Agent Management > Update unit
Q-Flow SupportBusiness Systems Q-Flow Support Sharepoint site is to be accessed when additional assistance is required.

SA(SM)/Managers are to ensure that CSO profiles are maintained under a Top Unit of their reporting CSC. Casual employees must also have a CSC as their Top Unit, not a Region. The Top Unit is to be managed and updated as needed by the SA(SM)/Manager when the CSO changes centre of operation.

Manager Profiles are to have the Top Unit as their region (unless approved otherwise) to ensure they can make changes to all profiles in the region. This includes changing a CSO to Manager and back as required. This type of request is not required to be sent to CSB Systems.
Refer below: Getting started > User Management

Note: Due to limited Licences available a quarterly review of Manager Profiles will be performed. All Regional offices will be required to send a list of the substantive SA(SM)'s and Managers in their region for Audit.
Getting Started
Signing into Q-Flow
Signing into Q-Flow
    Sign into Q-Flow by selecting the Q-Flow Icon on the desktop.

    On the Q-Flow Web Terminal select 'Sign in Automatically', this will automatically sign the operator into Q-Flow using their TMR Network connection.

    Once signed into Q-Flow, the operator will be directed to the 'Home Screen' as shown below:

Q-Flow Profile
Q-Flow Profile
    The Q-Flow profile must be accessed via the 'User' button on the left hand menu as shown below:



    Operators will be assigned the 'Current Unit', your CSC for example: Carseldine CSC when the original profile request is sent to Business Systems. Operators are only responsible for checking that they have been assigned to the correct unit, managers are responsible for updating "Current Unit" when operators move between centres in the region.

    Managers will need to update (via Update button) their 'Location' and 'Function' in Q-Flow:
    • 'Location' is usually the physical location for example: Counter 1, Counter 2, etc. Managers will need to set their location as 'Manager' so that tickets will not be called. Operators are able to update their own counter location within the CSC.
    • 'Function' is usually the group of services that the operator will provide. Managers will need to set their function as 'Manager' if they are not on a counter. Operators can change their function when directed by the manager.
Agent Management
Agent Management
    The Agent Management Console is a software component of the Q-Flow Systems. The Agent Management Console is where Managers will view the status of staff, queue depth and wait times within their centre. The Agent Management Console is accessed via the 'Management' button > 'Agent Management' on the left hand menu as shown below:


    Agent Management allows you to view and update the below details:
    • View operator information:
      • The operator
      • The functions they have been assigned
      • The counter they have been assigned to
    • View Q-Flow statuses:
      • Orange: If there is an orange face beside an operator it means they are logged in and active
      • Grey: If there is an grey face beside an operator it means they are logged in and idle
      • White: If there is a white face beside an operator it means they are not currently logged into Q-Flow
    • In Q-Flow you are able to for certain circumstances:
      • Update a staff members Current Unit (casual moving between centres)
      • Sign them out of Q-Flow (staff member on lunch/break and hasn't signed out)
      • Close the staff member out of a case (staff member on lunch/break and hasn't closed the case)

    Update Unit
    Update Unit
      To update an operator's unit:
      1. Select 'Management' on the left hand menu
      2. Select 'Agent Management'
      3. Click on the operator to update
      4. On the right hand side under 'Location and Function' select the Current Unit from the drop down menu
      5. Click the 'Update' button



      Note:Only managers are to be assigned the Region as their Top Unit

    Sign an operator out of Q-Flow
    Sign an operator out of Q-Flow
      To sign an operator out of Q-Flow:
      1. Select 'Management' on the left hand menu
      2. Select 'Agent Management'
      3. Click on the operator to be signed out
      4. On the right hand side under 'Sign Out' select the Sign Out Reason from the drop down menu
      5. Click the 'Sign Out' button

    Close an operator open case
    Close an operator open case
      If a case has been left open due to an operator not closing it, the system should automatically abort the case within 12 hours, if not, an email will need to be sent to Business Systems to request this.
    Online Service Statistics
    Online Service Statistics
      Online Service Statistics will display the wait times for customers that have taken a ticket from the kiosk and are waiting in the queue for each individual service.

      When a customer has been waiting in the queue for more than 7 minutes the wait time will change to orange and if the customer has been waiting in the queue for more than 8 minutes it will display as red.

      For appointment based transactions such as Practical Driving Tests and Interpreters the wait time will change to orange if the customer has been waiting for more than 5.5 minutes and then to red when a customer has been waiting for more than 8 minutes.

      Managers will be able to view the wait times for the centre in a number of locations in the Q-Flow system.
Info Centre
Info Centre
    The Info Centre is a software component of the Q-Flow System. Info Centre can provide a Manager with numerous reports, however these reports are not supported by TMR therefore Managers will still receive reports via COGNOS.




    Key reports in the 'Info Centre' are:
    • Unit Supervisor Dashboard

    • Service Level Analysis

    • Agent Performance Analysis
Location Management
Location Management
    Access 'Location Management' via the 'Management' button on the left hand menu as shown below:



    In 'Location Management' Managers will be able to:
    • Select the locations (counters) that should be serving Cash and EFTPOS or EFTPOS only
    • Select the locations (counters) that are the Accessibility Counters

    To make these changes take the following steps:
    1. Select 'Management' from the left hand menu
    2. Select 'Location Management'
    3. Select the centre to edit
    4. Select the counters that should be serving Cash and EFTPOS or EFTPOS only
    5. Select the counters that will be Accessibility Counters


    • Click on the 'Save' button to make the changes

User Management
User Management
    Access 'User Management' via the 'Management' button on the left hand menu as shown below:



    In 'User Management' a Manager will be able to:
    • move an operator to another unit (centre) within your region
    • change an operator from an Agent licence to a Manager licence
    If all Managers are unavailable for the day (for example they re all off due to unforeseen
    circumstances) the Agent can request for another Manager within their region to change their security group to Manager.

    Agents that are changed to a Manager licence must be changed back to an Agent licence once their normal duties resume.

    To move an operator to another unit:

    1.Select 'Management' from the left hand menu
    2.Select 'User Management'
    3.Enter the User ID of the operator in the Search Field
    4.Click the 'Search' button
    5.Click on the 'Move Unit' button
    6.Select the Top Unit where the operator is being moved to
    7.Click on the 'Save' button to make the changes.

    To move an operator to a manager licence take the following steps:

    1.Select 'Management' from the left hand menu
    2.Select 'User Management
    3.Enter the User ID of the operator in the Search Field
    4.Click the 'Search' button
    5.Click on the 'Edit' button
    6.Select 'Manager' from the User Security Group drop down list
    7.Click on the 'Save' button to make the changes.
    Note: Managers are unable to edit their own profile and will receive an warning message as shown below:

Reports and Dashboard
Reports and Dashboard
    Q-Flow has reports and dashboards available for SA(SM)/Managers to view.
    Reports
    Reports
      To access the Reports:
      1. Select 'Management' from the left hand menu
      2. Select 'Info Centre'
      3. Select the report to view



      Reminder: The reports provided by Q-Flow are not supported by TMR therefore Managers will still receive reports via COGNOS.
    Dashboard
    Dashboard
      The Dashboard can provide live data for a centre including the current average wait time.

      To access the Dashboard:
      1. Select 'Management' from the left hand menu
      2. Select 'Dashboard'
      3. On the Dashboard screen a particular location or 'All Locations' can be selected
      4. Click on the 'Home' button on the top right hand side of the page to exit




Signing Out
Signing Out
    To sign out of Q-Flow either click on the 'Sign Out' button at the top right hand corner of the screen or select 'User' from the left hand side menu and then click on 'Sign Out' as shown below:



    Q-Flow will prompt the operator (including Managers) to select a reason that describes why they are signing out of the system. Select the reason and then click the 'Sign Out' button.
