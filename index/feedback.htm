Feedback

 Requests for review of policy and/or procedural content in DocBase:
  • need to be discussed in the first instance with your SASM or Manager
  • if they are unable to resolve the issue, you or your supervisor should call Helpdirect or consult within your own region
  • if Helpdirect cannot resolve the issue, please progress to raising a policy or procedure BIID
  • the DocBase team do not have authorisation to review policy and/or procedural content, this type of feedback will only be addressed when submitted via appropriate channels.

 The DocBase3 feedback page is a feature that enables users to report minor issues such as spelling errors and broken links.

Only items listed in the ‘Feedback Category’ drop down list can be reported using this function.

Please access the feedback function from the page you want to provide feedback on, NOT the homepage. If you access the feedback function from the home page, you will need to provide a link to the relevant page in the ‘Comments/Feedback’ section below.
