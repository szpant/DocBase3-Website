SILAS Upgrade (7632)

Project NameSILAS Upgrade
When20 March 2021
OverviewSILAS is being upgraded to stabilise the system and to enhance the appearance. The updates will make the screens look and feel more modern, and the functions will be more user-friendly.

The changes to the way staff interact with SILAS will be minor. The key updates will include:
  • How you sign-in to SILAS.
  • A new 'splash' screen upon sign-in.
  • Screens enhanced to look more like front-end software (e.g. TICA) rather than a TRAILS-like view.
  • The 'ESC' key function will be replaced with the 'ALT' key. For example, instead of pressing 'ESC + 1', you will press 'ALT + 1'.
  • New 'Action Buttons' added so users have the option to select/click with their mouse as well as having keyboard functionality.
  • All documents will generate as a PDF and automatically open in a browser. From there, the document/certificate can be printed to any printer location.

Overall, screens will display the information and fields in the same location as the existing version of SILAS, but the look and feel will be improved.

The enhancements are in addition to the current functionality, meaning users can continue to use the existing Function keys and shortcuts.

Additional Information
Sign-in
Sign-in
    The new sign-in enhancements are for Windows 10 users. To launch SILAS after the upgrade you will go to your Windows Start Menu > S > SILAS 12c.

    The first time you log-in to SILAS after 20 March 2021, you'll need to re-set your password. You will receive a Welcome email with your Username and a temporary password so that you can re-set it. Your new password must be a minimum of 8 characters and contain at least one uppercase letter, one lowercase letter, and one number.

    Your SILAS password is a separate log-in to your TMR log-in. If you experience issues with your password or require another re-set, you will need to request a SILAS password re-set via Service Centre Online.

    Note:Launching SILAS via the existing access will be removed once the upgrade is live.
Function keys and TAB key
Function keys and TAB key
    • The Function or "F" keys can still be used. Alternatively, the mouse can be used to click on buttons/selection.
    • The TAB key can still be used. Alternatively, the mouse can be used to move between fields.
Printing
Printing
    The need for SILAS specific printers has been removed, meaning users will have greater flexibility when it comes to printing documents. SILAS documents/certificates will now generate automatically as a PDF and print to any printer location. The certificate templates are no longer needed, as the authority details and certificate will print as a whole document.

    For 'print to PDF', you need to:
    • add your document parameters as per the current process
    • print the document to any printer location (i.e. Kyocera, back-office printer/s)

    Note:The PDF document will only be available to view and print in SILAS during the current session you are logged in to. During the session you may go to Report Manager and view and reprint any document/s generated during that session. However, once the session is closed the documents will no longer be available to reprint, and must be printed by logging into a new session.
New Users
New Users
    Service Centre Online (SCO) will be used to set up or amend a SILAS account, rather than the existing procedure of emailing System Security.

    The SILAS Access Application (F3674) form needs to be completed, and then attached to the SCO request, 'SILAS User Administration'.
System or Processing Issues
System or Processing Issues
    System issues
    System issues
      The following system issues can be escalated by raising a SCO request > SILAS User Administration:
      • New access
      • Modify access
      • Remove access
      • Password reset
      • Software installation (when an account already exists)

      System issues outside of these reasons may be escalated via the standard escalation path, i.e. Help Direct > Capability, Engagement and Learning.

    Processing issues
    Processing issues
      For processing and/or printing procedures:
      Refer: Industry > 2. Dangerous Goods > The How To: Dangerous Goods > Dangerous Goods Procedures > Dangerous Goods Vehicle Licence
      and
      Refer: Industry > 6. Tow Trucks > The How To: Tow Truck > Tow Truck Procedures

      Escalate processing issues via the standard escalation path, i.e. Help Direct > Capability, Engagement and Learning.
