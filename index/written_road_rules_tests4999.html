About Written Road Rules Tests
What it isA written road rules test tests a person's knowledge of the Queensland Road Rules.

When a Written Road Rules Test is Required
When RequiredAn applicant for a Qld licence may be required to undertake one or more test types, depending on the class being applied for and the applicant's licence/class history:
Australian Licence Holders/Applicants
NZ, Recognised and Experienced Driver Recognition Country Licence Holders
NOT Recognised and Experienced Driver Recognised Country Licence Holders
Returning Drivers
When a Written Road Rules Test is NOT Required
When NOT RequiredA test is not required if the person, within 5 years before this application meets any of the following:
Passed a TestPassed a road rules test in any Australian State or Territory for the same class of licence applied for
Aust LicenceHeld an Australian driver licence for the same class of licence applied for
Police OfficerA Police Officer who has presented a Licence Endorsement Request for the same class being applied for
NZ LicenceHeld a class C, RE or R New Zealand driver licence and applies for the same class of Qld licence
Foreign Licence from Recognised and Experienced Driver Recognition Countries Held a class C, RE or R Foreign licence that is one from a country listed on:
Austroads Approved Countries

Note:The PrepL or Written Test prerequisite is to be exempt in TICA.

Diplomatic/Consular and TECO Staff Transfers

Notes:An Australian, New Zealand or Foreign driver licence includes a Learner licence.

A test is not required for a current serving member of the ADF who holds a current ADF driver licence
