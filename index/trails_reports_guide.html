TRAILS Reports Guide (5015)


Prioritisation of Reports
First priority: Exemption Reports
Second Priority:Manual Adjustment Report
Third Priority:Customer debit/credit report
Printing of ReportsTrails reports are received via Outlook Mail Inbox and sent to either:
  • CSC/business unit
  • Specific user

Reports can be accessed by any of the nominated users. Nominated users are identified by the CSC Managers who forward the request to Service Centre Online.
Identifying ReportsReports can be identified as follows
  • A paper clip symbol will be displayed on the left hand side of the listing to indicate an attachment
  • The sender will be listed as TRAILS@TRPT.qld.gov.au
  • The subject will be the report name
Modification Books and Plates ReportEvery Wednesday a 'Modification Books and Plate Sales' report is generated and emailed to CSCs that have sold Modification Books or Plates in the last week. This report should be checked to ensure that each application form and any supporting documentation has been scanned and emailed to AP Administrator.
Redirection of QPS ReportsFor audit purposes the following TRAILS reports have been redirected (by applying the FILE involved party qualification to the QPS involved party record) to a nominated supporting CSC.

Redirected reports include:
  • RG61 Document Filing Summary
  • AA62 Daily Customer Account Report
  • AA92 List of Operational Account Balances
  • AA93 List of Manual Adjustments
  • AA97 Exception Report of Payment Requests Acquitted by accounts

Reports are actioned as per current guidelines including the use of the Follow up register.

The redirection is managed through the FILE involved party qualification. Changes to where a QPS station is redirected must be approved by the Manager (CSD) within the region.

    Adjustment to Payment Item
      The report is generated for all payment item adjustment at any receipting office/location.

      It is generated daily or on demand for a specified period.

      Details Contained
      Details Contained
        • Adjustment date
        • User Id and user name
        • Adjustment type, amount and number
        • Payment item number
        • Payment item text
        • Location/receipting office name
      Action Required
      Action Required
        CSC Manager/Supervisor/ Banking Officer is required to:
        • Ensure report is printed daily
        • Confirm adjustments by checking the Maintain Payment Item screen
        • Sign and date the report
        • File and retain the report

    Password Reset Notification
    Password Reset Notification
      This report lists all TRAILS operator passwords reset by an authorised officer.

      The Report
      The Report
        • Is produced after the end of day procedures
        • Forms part of the Controlled Self Assessment for Service Delivery staff
      Details Produced
      Details Produced
        • Date and time the password was reset
        • Userid of the operator whose password has been reset
        • Userid of the authorised officer who reset the password
      Action Required
      Action Required
        Authorised officer is required to:
        • Ensure the report is printed daily
        • Attach a copy of the:
          • Operator request to reset password
          • Advice to operator that password has been reset
        • Arrange operator to sign the report in the 'Verified By Signature' area to confirm the action was correct
        • File and retain reports
      Filing of Report
      Filing of Report
        To assist in performing an audit the report should be filed seperately within the office in a secure but accessible location. Internal Audit Services recommends these reports are kept for at least 12months in the office, then archived where they are to be retained for 7 years.
