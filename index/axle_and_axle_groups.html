Axle and Axle Groups (3059)

Axle Configuration
Refers to the grouping of axles on a vehicle.

Vehicle that require an axle configuration include:
  • Motor vehicles with a GVM >4.5t
  • Trailers with an ATM >4.5t
TICA Axle Configuration Codes
TICA Axle Configuration Codes
    eg 22L0.6-4400L0.6/4.9

    CodeDefinition
    22 Number of axles and number of tyres fitted to the axles in the first axle group
    eg 2 axles with 2 tyres on each axle
    LSuspension type
    L = Load sharing
    NL = Non load sharing
    0.6Distance between axle centres of the first group if there is more than 1 axle in the group
    -Separates the axle groups
    4400Number of axles and number of tyres fitted to the axles in the second axle group
    eg 4400 is two axles with four tyres on each axle
    LSuspension type of second axle group
    L = Load sharing
    NL = Non load sharing
    0.6Distance between axle centres of the second group if there is more than 1 axle in the group
    /Separates the axle configuration from the extreme axle distance
    4.9Extreme axle distance
