30/04/2021 - Changing a name in TICA
30/04/2021 - Changing a name in TICA

    When recording a name change in TICA, if the Reason CORRECTION is selected, please ensure that a new Image Set (Facial and Signature) is captured. If it’s not captured, the existing Image Set becomes invalid and the customer can’t transact online, such as applying for a learner licence through PrepL.

    If an individual’s name needs to be corrected and the customer doesn’t want a new Image Set, only an Operator with a DAMANGR profile can process the name change in TICA. The DAMANGR must select the reason CORRECTION – EXEMPT IMAGE CAPTURE which will allow the existing image set to be used, and the customer will be able to use online services.

    Lee Boyle
    Principal Advisor (Identity Management)
    Identity Management Unit

27/04/2021 - Defective Work Diaries
27/04/2021 - Defective Work Diaries
    You were previously advised that there was a batch of potentially defective work diaries that needed to be put aside while the issue was reviewed.

    It has been determined that the issue does not specifically relate to that batch so the diaries can be used.

    As the issue is not batch related we need to check each diary when it is issued to ensure the writing on the top page is transferring through to the copy pages.

    Do this by checking, when you write in the issuing centre, that the writing is legible on the copy pages.

    If the writing does not transfer through:
    • do not issue the diary,
    • put the diary aside and keep it until further advice provided
    • send an email to Compliance_Support detailing the book number

    If you are holding defective diaries found previously please continue to keep them and email the book number to Compliance_Support.

    If a customer presents a defective diary:
    • Email the book number to Compliance_ Support, noting that it was a customer return
    • On the application page in the defective diary mark it as “Cancelled”, date and stamp with CSC stamp if available
    • Return the defective book to the customer, advising them they must keep the defective diary for their records.
    • Issue a free replacement.

    Philippa Smith

23/04/2021 - Valid image requirements for External Agency Cards

22/04/2021 - Possible Rebirthing of Toyota Landcruisers - Update
22/04/2021 - Possible Rebirthing of Toyota Landcruisers - Update

    In reference to the message sent on 17 February 2021 about a noted trend in the theft of (or attempted theft of) Toyota Landcruiser’s from holding yards/businesses in North Brisbane; staff are advised that any Landcruisers presented for registration for the first time in Qld (other than new vehicles) be inspected to ensure the stamped VIN or Chassis number matches the compliance plate.

    If the stamped VIN or Chassis number does not match the compliance plate, please ensure you follow the current process in DocBase for surrogate VINs. Refer: Vehicle Rego > 05. Vehicle Requirements > Vehicle Identifiers and Descriptors > Surrogates

    For all enquiries, please email: policyproceduressupport@tmr.qld.gov.au

20/04/2021 - Use of Kyocera printer data ports to power USB devices (desk fans)
20/04/2021 - Use of Kyocera printer data ports to power USB devices (desk fans)

    Data ports on Kyocera printers must not be used to power USB devices, for example desk fans, as this may cause the printer ports to blow.

    Any USB devices should be disconnected from printers ASAP.

09/04/2021 - Heavy Vehicle Registration Assessment Scheme (HVRAS) vehicle inspection sheets
09/04/2021 - Heavy Vehicle Registration Assessment Scheme (HVRAS) vehicle inspection sheets
    HVRAS vehicle inspection sheets are now screen fillable PDF forms. This means the user has the option of directly typing the required information into each field on the HVRAS vehicle inspection sheet prior to printing and signing.

    This option has been implemented as a result of customer feedback to make the completion of HVRAS vehicle inspection sheets more efficient for users and therefore, improving customer experience.

    A HVRAS accredited person can also continue to complete HVRAS vehicle inspection sheets by hand.

    A HVRAS accredited person's original (wet) signature is still required on the HVRAS vehicle inspection sheet. Photocopied or digitised signatures on HVRAS vehicle inspection sheets should not be accepted.

    If you have any further enquiries, please contact HVRAS Policy on 3066 2496 or email: HVRAS_Policy@tmr.qld.gov.au.

    Maria Stevenson
    Principal Policy Advisor
    Operator Accreditation and Authorisation
