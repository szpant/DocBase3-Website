CSC Identity Document Verification Failure Report
CSC Identity Document Verification Failure Report
    What it isThe lists each EOI document with a status of 'pending'.
    When Produced
    The Report is Produced
    The Report is Produced
      • After the end of day procedures
      • As a control measure
      • Daily
    Details
    Details Produced
    Details Produced
      • User ID
      • A list of documents that failed verification or those that have been identified as a duplicate match.
    Req'ments
    Action Required
    Action Required
      CSC Manager/SA(SM) is requried to:
      1.Ensure the report is printed daily
      2.Ensure that there is a record of a fax/email for each customer on the report list.
      3.A separate file / folder should be kept that has copies of documents forwarded to IMU for investigation. Use this file or folder to confirm that each fax matches a name on the report.
      4.If not, ensure that the missing paperwork is now faxed or emailed. Look in the filing for the respective paperwork.

      Note:Report must be actioned by an officer who was not the original author of any transaction(s).
      Filing of Report
      Filing of Report
        Once administrative actions have been taken the report may be destroyed.
    IMU Identity Document Verification Failure Report
    IMU Identity Document Verification Failure Report
      What it isThe report lists documents that are duplicate and/or have failed verification as well as those documents that are currently under investigareport tion by the unit.
      When Produced
      The Report is Produced
      The Report is Produced
        • After the end of day procedures
        • As a control measure
        • Daily
      Details
      Details Produced
      Details Produced
        • User ID
        • A report is required to list EOI documents that require further action due to:
          • verification failure, or
          • having been detected as exiting more than once in TICA (duplicate matches),
          or
          • being currently under investigation.
      Req'ments
      Action Required
      Action Required
        Officer is requried to:
        1.Ensure the report is printed daily
        2.Ensure that there is a record of a fax/email for each customer on the report list.
        3.A separate file / folder should be kept that has copies of documents faxed / emailed to the Unit. Use this file or folder to confirm that each fax matches a name on the report and that this has been checked for known anomalies.
        4.Also confirm that this has been forwarded on to the issuing authority for verification.
        5.If this is OK proceed to 6.
        6.If not, ensure that the missing paperwork is now faxed or emailed.
        7.If documents have not been received contact the relevant CSC/Agent and request the documents.
        8.Sign and date the report and record USERID
        9.File and retain the report

      Filing of Report
      Filing of Report
        To assist in performing an audit the report should be filed seperately within the office in a secure but accessible location. Internal Audit Services recommends these reports are kept for at least 12 months in the office, then archived where they are to be retained for 6 years.
