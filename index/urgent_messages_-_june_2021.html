30/06/2021 - NAAAN Format Conversion
30/06/2021 - NAAAN Format Conversion

    On 26 June 2021, personalised plates with a combination of NAAAN were aligned with their respective colour specifications in TICA. Approximately 20,000 records were updated.

    Along with this update, a business rule was introduced that prevents any invalidation of these records prior to 26 June 2021. Should a record require invalidation, a SCO will now need to be raised to request a datafix of the record.

    It is not envisaged that a large volume of these records will be affected.

    Apologies for any inconvenience caused.

25/06/2021 - Procedure to Follow Before Signing into TICA on Monday 28 June 2021
25/06/2021 - Procedure to Follow Before Signing into TICA on Monday 28 June 2021

    Registration and Licensing Systems are conducting an application release this evening, 25 June 2021. From time to time, changes impacting TICA have triggered intermittent issues with TICA post deployment.

    To avoid these issues, we advise you carry out the following procedure before logging into TICA on Monday 28 June 2021.

    It is recommended to clear your cache (in Internet Explorer).

    To clear your cache please follow the below process:

    1. Open Internet Explorer (Please note that it must be Internet Explorer and not Edge. To find Internet Explorer, click on the windows button in the start bar and type in Internet Explorer)

    2. Select 'Tools' from the menu line (or alt X on keyboard when you are in IE) and then 'Internet options'



    3. Choose to 'Delete' Browsing History



    4. Please uncheck the ‘Preserve Favourites website data’ box and then check the boxes as per below in the example. Once completed click ‘delete’ and this will return you to the Internet Option page and click ‘OK’.



    5. Repeat step 2 and select 'Tools' from the menu line (or alt X on keyboard when you are in IE) and then 'Internet options'



    6. Click 'Settings' button



    7. Click 'View files' button as below



    8. Upon clicking 'View files' the windows explorer will launch with directory location on your local machine containing the temporarily cached files. Please verify the directory location looks like the below screenshot " <<logged in username>>\AppData\Local\Microsoft\Windows\INetCache"



    9. Select All temporary files within the above directory location. You can use your mouse to select all files or you can select one file on the right and click "Ctrl +A" combination to select all files. Once all files within the directory are selected, please click 'Delete' key on your keyboard to delete the files. You will be prompted with a Yes/No dialogue box to confirm, please verify the message looks similar to below screenshot and click 'Yes' to delete the temporary files.



    10. Close ALL IE windows, then only launch TICA

    If, after following these instructions, you are still unable to access TICA, please contact the Service Centre on 3066 2000.

    Lindsay Abdi
