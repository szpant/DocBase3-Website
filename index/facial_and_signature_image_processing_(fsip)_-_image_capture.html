Facial and Signature Image Processing (FSIP) - Image Capture (7624)

Project NameFacial and Signature Image Processing – Image Capture
WhenDecember 2020 TICA release with phased rollout from December 2020 – June 2021
BackgroundTMR captures customer facial, signature and document images to support the issue of Driver Licence, Marine Licence, Industry Authority and Photo Identification cards.

The functionality that enables these image capture processes is being replaced with a new web-based solution. This means the process to capture the image will occur outside of TICA in an internet browser window.

To enable the new web-based solution to be utilised, a new version of TICA must be installed. This will be deployed to 'Software Centre' on all relevant desktops in a phased approach. Operators will be notified via their SASM when this is occurring for their location.
Who will be impacted and how?
CSC Operators
CSC Operators
    • Will need to install the latest TICA software when advised by their SASM.
    • Will notice minor changes when the image capture process:
      • Is loading - a webpage dialog box will populate which will not require any action.
      • Has been completed or cancelled - another webpage dialog box will populate and the operator will need to press "Continue" to return to TICA.

    Note: Once the operator presses Continue, the existing TICA messages will display based on what occurred in the image capture webpage (for example, Image Exception – Contact Identity Management Unit).

    If issues arise after the new TICA software is installed, refer to the Known FSIP issues link for instructions. If unable to resolve, contact the FSIP project at FSIP_Project@tmr.qld.gov.au
Remote area kit locations
Remote area kit locations
    • Will now be able to scan facial, signature images and document images directly into TICA (they no longer need to save the file first).
IDLU
IDLU
    • Will now be able to save images to a secure "vault" location on their local laptop (rather than saving to USB) and then upload these images to TICA when they are back in the office.

    Note: The vault is a piece of software that is loaded onto specific machines. Once the images are uploaded to TICA, they are automatically deleted from the vault.
Further information
  • Only desktops with Windows 10 will be able to download the new software.
  • No changes are being made to any of the hardware (for example, cameras/scanners/printers).
  • A separate activity will be undertaken to roll out new signature capture devices which may occur after the new software has been installed.
  • SASMs will be notified in the weeks prior to their centre being rolled out.

Troubleshooting
Remote Area Kit Image Failures
Remote Area Kit Image Failures
    If staff are experiencing problems when capturing images from remote kits, the below tips on image manipulation may assist with capturing a compliant image:
    • zoom the PDF to 300-400%, then crop the image to save.
    • upload a larger image
    • ensure the image is cropped as close to the border of the passport photo as possible
    • rotate the image, if not straight
    • select a different supplied photo
