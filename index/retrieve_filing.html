Retrieve Filing (2803)

Applies to
Applies to
    Retrieving Documents/Filing due to:
    • Show Cause Procedures (originals)
    • Verification/Correction of records (copies)

    If the customer's record is suppressed Refer: QT Security
1
Determine document type
Determine document type
    • Determine what type of filed document is to be retrieved:
      • Copy (verification/correction of records)
      • Single original document (Manager email approval required)
      • Box of original documents (Manager email approval required)
2
Obtain Document Storage details
3

Access Iron Mountain Request Form

    System Security Unit

4
Original Document Requests
Original Document Requests
    • Send an email to Manager requesting approval for the original document (single or box)
    • File the Manager's approval response in a shared mail box (e.g. the CSC Mail box)
    • For original box requests:
      • Box will be delivered by Iron Mountain to COS
      • COS will store the box in a secure location (mailroom)
      • COS officer will promptly review the box contents and redact with permanent black marker any payment card details*
      • COS will keep a record (e.g. register) of all original boxes received
      • COS will forward the box onto the area that requested the box by courier.
      *Note:COS may request resourcing assistance for the redacting process from the requesting location if appropriate.
